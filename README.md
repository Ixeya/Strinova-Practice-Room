基础操作 

1. 基础音高与八度

中音区：直接写数字 1 到 7

半音 (黑键)：前面加 #，例如 #4

高低八度：低音加点 .（如 6.），高音加单撇 '（如 3'），倍高音加双撇 ''（如 1''）。

2. 节奏控制：灵魂的“先加后乘”魔法
默认情况下，每一个单独的数字就是 1拍 (四分音符)。

加时 (延长)：使用 -。每个 - 增加 1 拍。

1- = 2拍 (二分音符)

1-- = 3拍

减时 (缩短)：使用 ^。每个 ^ 会将整体时间砍半！^ 写在音符前面或后面都可以。

^1 或 1^ = 0.5拍 (八分音符)

^^1 = 0.25拍 (十六分音符)

附点音符 (进阶技巧)：利用系统的数学逻辑可以组合出复杂时值！

0.75拍 (附点八分音符)：写成 ^^1--。引擎计算：(1基础 + 2延长) * 0.5 * 0.5 = 0.75拍。

1.5拍 (附点四分音符)：写成 ^1--。引擎计算：(1基础 + 2延长) * 0.5 = 1.5拍。

3. 休止符与空白

停顿：直接写一个单独的 - 代表休止 1 拍。如果需要休止半拍，写成 ^- 即可。

空格和换行只是为了排版好看，引擎会自动跳过它们。

4. 多键和弦

使用 & 将多个音符连在一起，引擎会瞬间同时触发它们！

示例：1'&3'&5' (同时弹奏高音1、3、5)。

和弦同样可以带节奏修饰符，例如：^1&3&5 代表一个 0.5 拍的短促和弦。

5. 变速 (BPM)

在任意位置输入 #数字# 即可实时改变曲速。例如 #156# 会将 BPM 瞬间切换到 156。

## Task
我将提供一张或多张简谱/五线谱的图片。你需要严格遵守我自定义的“引擎解析语法”，将图片中的音符精准转化为纯文本格式。不要猜测、不要脑补乐理，仅做精确的视觉识别与格式转换。

## Engine Logic & Syntax Rules (必须严格遵守)

### 1. 音高与八度 (Pitch)
- **中音区 (基础)**: `1`, `2`, `3`, `4`, `5`, `6`, `7`
- **低音区 (下方带点)**: `1.`, `2.`, `3.` ...
- **高音区 (上方带点)**: 单个点用单引号 `1'`, 两个点用双引号 `1''`, 三个点 `1'''`
- **半音/变化音**: 在音符最前面加 `#`。例如：`#4`, `#1'`, `#5''`

### 2. 节奏与时值 (Rhythm Math - 核心算法)
引擎采用“先加法，后乘法”的数学计算逻辑。**单个数字本身 = 1.0 拍。**
- **延长符 (`-`)**: 每个 `-` 代表增加 1.0 拍。(加法)
- **缩短符 (`^`)**: 每个 `^` 会将当前总拍数乘以 0.5。(乘法)
**常用时值公式（请直接套用）：**
- 1拍 (四分音符) = `1`
- 2拍 (二分音符) = `1-`
- 0.5拍 (八分音符/单下划线) = `^1`
- 0.25拍 (十六分音符/双下划线) = `^^1`
- 1.5拍 (附点四分音符) = `^1--`  (计算逻辑：(1+2)*0.5 = 1.5)
- 0.75拍 (附点八分音符) = `^^1--` (计算逻辑：(1+2)*0.5*0.5 = 0.75)

### 3. 休止符 (Rests)
引擎使用 `-` 替代传统的 `0`。
- 1拍休止符 = `-`
- 0.5拍休止符 = `^-`
- 0.25拍休止符 = `^^-`

### 4. 和弦与多声部 (Chords)
同时按下的多个音，使用 `&` 连接。节奏修饰符可以写在整个和弦的前后。
- 例如同时弹奏高音1和中音3，持续 0.5 拍：`^1'&3`
- 例如双声部大和弦持续 1.5 拍：`^1''&5'&1'--`

### 5. 曲速 (BPM)
在乐谱的最开头，使用 `#数字#` 标记曲速。例如：`#120#`

## Formatting constraints
1. 每一小节内部的音符用**空格**隔开。
2. 保持排版清晰，可以按乐句换行。
3. **输出要求**：不要给我任何多余的解释、废话或乐理分析，直接输出符合上述语法的纯代码块文本。

# 1. 主解析器：将文本转换为带坐标的事件序列 (Tokenization)
func _parse_score_with_positions(text: String) -> Array:
	var events = []
	var current_line = 0
	var current_col = 0
	var i = 0
	var length = text.length()
	var pending_prefix = ""
	
	while i < length:
		# 跳过空白字符并记录光标位置
		while i < length:
			var c = text[i]
			if c == "\n":
				current_line += 1
				current_col = 0
				i += 1
			elif c == " " or c == "\t" or c == "\r":
				current_col += 1
				i += 1
			else:
				break 
		
		if i >= length: break
		
		var start_line = current_line
		var start_col = current_col
		var start_index = i
		
		# 读取连续的 Token (直到遇到空格或换行)
		while i < length:
			var c = text[i]
			if c == " " or c == "\t" or c == "\n" or c == "\r":
				break
			i += 1
			current_col += 1
			
		var token = text.substr(start_index, i - start_index)
		
		# 识别 BPM 变速标记 (例如 #120#)
		if token.begins_with("#") and token.ends_with("#") and token.length() > 2:
			pending_prefix = "" 
			var bpm_str = token.substr(1, token.length() - 2)
			if bpm_str.is_valid_int():
				events.append({
					"type": "bpm",
					"value": bpm_str.to_int(),
					"line": start_line,
					"col": start_col,
					"length": token.length()
				})
		else:
			# 校验非法字符
			if not _is_valid_note(token):
				pending_prefix = "" 
				continue 

			# 处理前置修饰符拼接 (允许 ^ 写在音符前面)
			if token.replace("^", "") == "":
				pending_prefix += token
				continue
			else:
				var full_text = pending_prefix + token
				pending_prefix = "" 
				
				events.append({
					"type": "note",
					"text": full_text, 
					"line": start_line, 
					"col": start_col,
					"length": token.length() 
				})
			
	return events

# 2. 时值计算器：引擎的灵魂算法 (先加法，后乘法)
func _calculate_beat_duration(token: String) -> Dictionary:
	# 提取纯音符本体
	var check_text = token.replace("^", "").replace("-", "").replace("~", "").replace("&", "")
	var has_note_body = check_text.strip_edges().length() > 0
	
	# 基础时长：有音符就是 1.0 拍，纯休止符是 0.0 拍
	var duration = 1.0 if has_note_body else 0.0
	
	# 加法逻辑：每个 '-' 增加 1 拍
	var dash_count = token.count("-")
	if dash_count > 0: duration += float(dash_count)
	
	# 乘法逻辑：每个 '^' 将当前总拍数砍半
	var caret_count = token.count("^")
	if caret_count > 0:
		for i in range(caret_count): duration *= 0.5
			
	# 返回计算后的时长，以及剥离了修饰符的干净音符文本
	var clean_text = token.replace("^", "").replace("-", "").replace("~", "")
	return {"duration": duration, "clean_text": clean_text}

# 3. 语法白名单检查
func _is_valid_note(token: String) -> bool:
	if token == "": return false
	# 允许的字符：数字、缩短符^、高低八度'和.、延长符-、延音~、和弦&、升降号#
	var allowed = "0123456789^'-~\"&#."
	for i in range(token.length()):
		if not allowed.contains(token[i]):
			return false 
	return true
